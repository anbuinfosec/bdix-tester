# BDIX-TESTER

## BDIX-TESTER is a python tool for check all working servers on your isp.

## INFO
* CLEAN UI.
* TOTAL **138** SERVERS.
* FAST REQUEST.

## REQUIREMENTS
* python3
* python-pip
* requests
* pytz

## INSTALL

````
pkg update && upgrade -y
pkg install python3 -y
pkg install git -y
git clone https://github.com/anbuinfosec/bdix-tester
cd bdix-tester
pip3 install -r requirements.txt
python3 bdix.py
````
